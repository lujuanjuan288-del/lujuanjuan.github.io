# 酒店民宿模块化长图编辑器

## 打开方式

在浏览器打开：

`http://localhost:4321/index.html`

如果没有启动本地服务，也可以直接双击打开 `index.html`。少数浏览器对本地文件导出图片有限制，遇到这种情况建议使用本地服务方式。

## 推荐制作流程

1. 先选择“田园自然”或“城市人文”整体主题。
2. 设置酒店名称、主标语和地点信息。
3. 新增图文、多图或客房板块。
4. 点击板块后调整顺序、复制、删除和图片布局。
5. 点击图片位后替换图片，并调整宽高、比例、圆角和裁切焦点。
6. 点击画布文字后调整字号、颜色、对齐、粗体或竖排。
7. 保存项目，最后导出 PNG 或 PDF。

## 板块功能

- 新增图文板块、多图板块和客房板块。
- 选中板块可以上移、下移、复制和删除。
- 支持单图通栏、左右双图、三列、四宫格、左大右小、上大下双图、圆形组合和自由拼贴。
- 可以调整每个板块的上下留白。
- 可以设置纯色、上下渐隐、主题色带或透明背景。

## 自由拼贴

1. 先点击需要编辑的板块。
2. 将“板块图片布局”改为“自由拼贴”。
3. 直接拖动图片改变位置。
4. 拖动图片右下角控制点调整宽度和高度。
5. 使用旋转、透明度和图层按钮制作错落叠放效果。
6. 通过“拼贴画布高度”控制该板块占用的纵向空间。

从自由拼贴切回普通布局后，图片会重新进入规则网格。

## 图片功能

- 单张添加或批量添加图片。
- 替换和删除选中图片。
- 调整图片宽度、高度、圆角。
- 设置正方形、16:9、3:4、圆形或自定义矩形。
- 设置铺满裁切或完整显示。
- 调整横向和纵向裁切焦点。
- 自由拼贴模式下支持拖动、缩放、旋转、透明度和图层顺序。

## 装饰元素

- 支持装饰线、圆环、编号标签和植物符号。
- 装饰元素可以拖动、缩放、旋转和调透明度。
- 可以调整颜色和图层顺序。
- 装饰元素会跟随板块保存，并正常进入最终导出图片。

## 项目管理

- 编辑内容会自动保存到当前浏览器。
- “保存到本机”可以立即保存。
- “导出项目文件”会生成包含图片、文字和布局的 JSON 文件。
- 其他用户可通过“导入项目文件”继续编辑。
- “复制工具链接”用于分享已部署的网页地址。
- 支持撤销、重做，也支持 `Ctrl/Command + Z`。
- “恢复初始模板”会清除当前浏览器中的已保存项目。

## 手机使用

- 手机打开时，编辑面板默认收起。
- 点击左下角圆形菜单按钮打开或关闭编辑面板。
- 长图画布会按照屏幕宽度缩放，实际导出尺寸不受影响。

## 网页发布

项目已经包含以下发布配置：

- GitHub Pages：`.github/workflows/pages.yml`
- Vercel：`vercel.json`
- Netlify：`netlify.toml`
- PWA 离线使用：`manifest.webmanifest` 和 `sw.js`

发布后，任何人都可以通过网址使用工具。项目内容默认只保存在各自浏览器中，不会自动上传到服务器。

## 导出方式

- 点击“导出 PNG 长图”保存成长图图片。
- 点击“打印 / 保存 PDF”可通过浏览器打印面板另存为 PDF。
- 需要只看长图本体时，可打开：

`http://localhost:4321/index.html?preview=1`

## 建议图片比例

- 首图外观：竖图或横图都可，建议清晰大图。
- 双图模块：建议使用 4:3 或 1:1。
- 横向大图：建议 16:9 或 3:2。
- 客房模块：建议准备主卧、浴室、窗景、床品、公共区等 8-12 张图片。
- 完整长图建议准备 20-35 张经过筛选的实拍图。
